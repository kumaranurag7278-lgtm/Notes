📌 Part 1: Plotly Express Basics
Q1. What is the basic syntax?

Try to answer before looking.

Answer

import plotly.express as px

fig = px.scatter(df, x="A", y="B")
fig.show()

Remember the pattern:

Import
↓

Choose chart

↓

Pass DataFrame

↓

Specify columns

↓

show()
Q2. Why is Plotly Express so popular?

✅ Less code

✅ Works directly with Pandas

✅ Interactive graphs

📌 Part 2: Important Arguments
color
px.scatter(df,
           x="A",
           y="B",
           color="Species")

Purpose

Different categories → different colors.

size
size="Population"

Purpose

Marker size depends on a column.

hover_data
hover_data=["Country","GDP"]

Purpose

Shows extra information when you hover.

📌 Part 3: Chart Types

Know when to use each one.

Chart	Use
px.scatter()	Relationship between two numerical variables
px.line()	Trends over time
px.bar()	Compare categories
px.histogram()	Distribution
px.box()	Outliers & spread
px.imshow()	Heatmaps (e.g., correlation matrix)

This table is worth memorizing.

📌 Part 4: Customization
Layout
fig.update_layout(
    title="Title",
    xaxis_title="X",
    yaxis_title="Y"
)

Changes

title
axes
legend
overall layout
Traces
fig.update_traces(
    marker_size=12
)

Changes the plotted data itself.

📌 Part 5: Faceting
facet_col
facet_col="Species"

Creates one graph per category horizontally.

facet_row
facet_row="Species"

Creates one graph vertically.

facet_col_wrap
facet_col_wrap=3

Wraps many plots into multiple rows.

📌 Part 6: Graph Objects

Basic syntax

import plotly.graph_objects as go

fig = go.Figure()

fig.show()

Unlike Plotly Express,

you start with an empty figure.

📌 Part 7: Adding Traces
fig.add_trace(
    go.Scatter(...)
)

Think of a trace as one chart layer.

Example

Trace 1 → Sales

Trace 2 → Profit

Trace 3 → Expenses

One figure can contain many traces.

📌 Part 8: Scatter Modes
mode="markers"

Only points.

mode="lines"

Only lines.

mode="lines+markers"

Both.

This is the most commonly used mode.

📌 Part 9: Subplots

Create layout

from plotly.subplots import make_subplots

fig = make_subplots(
    rows=2,
    cols=2
)

Add chart

fig.add_trace(
    go.Bar(...),
    row=1,
    col=2
)

Remember:

row= and col= decide where the chart goes.

📌 Part 10: When to use Express vs GO
Plotly Express
Fast
Few lines of code
EDA
Simple visualizations
Graph Objects
Multiple traces
Dashboards
Complex plots
Full customization

A practical rule:

Start with Plotly Express. If you can't achieve what you want, switch to Graph Objects.
